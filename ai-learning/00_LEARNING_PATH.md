# AI/ML Learning Path for K8s Agent Project
## Your Journey from Zero to Confident ML Engineer

**Goal:** Master AI concepts needed to build and explain a production-grade K8s troubleshooting agent

**Timeline:** 8-12 hours of focused study before starting project

**Outcome:** Be able to confidently explain every technical decision and compare alternatives
