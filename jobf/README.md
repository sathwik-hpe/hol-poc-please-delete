## üí° Tips for Success

### During Preparation:
1. **Hands-on > Reading**: Build things, don't just read
2. **Teach Others**: Best way to solidify knowledge
3. **Document Everything**: Write as you learn
4. **Use Real Tools**: AWS Free Tier, not simulators
5. **Join Communities**: AWS Slack, Terraform forums

### During Interview:
1. **Ask Questions**: Clarify before answering
2. **Think Aloud**: Show your thought process
3. **Use Examples**: From your real experience
4. **Be Honest**: Admit gaps, show learning agility
5. **Stay Calm**: Take a breath, collect thoughts

### Red Flags to Avoid:
‚ùå "I just use the AWS console"  
‚ùå "I'm still learning Python"  
‚ùå "I haven't used Terraform much"  
‚ùå "I prefer manual configuration"  
‚ùå Blaming others in behavioral questions
